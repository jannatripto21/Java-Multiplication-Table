# Java Multiplication Table 📊

A simple Java application that generates the multiplication table of a given number.

## 🚀 Features

- User input support
- Generates multiplication table from 1 to 10
- Beginner-friendly Java implementation
- Uses loops and arithmetic operations
- Console-based application

---

## 📖 Example

### Input

```text
Enter a number: 5
```

### Output

```text
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
```

---

## 🛠 Technologies Used

- Java
- Scanner Class
- For Loop

---

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/JavaMultiplicationTable.git
```

### Go to project directory

```bash
cd JavaMultiplicationTable/src
```

### Compile the program

```bash
javac MultiplicationTable.java
```

### Run the program

```bash
java MultiplicationTable
```
